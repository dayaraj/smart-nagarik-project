BUILD AND DEPLOY NODEJS APPS WITH MICROSOFT AZURE

Innovative Computer Engineering Students' Society (i-CES) brought three day workshop to build and deploy Node.js Apps with Microsoft Azure. You will learn to build a web application using Node.JS, a javascript framework. You will also learn how to deploy your web app to Microsoft Azure for your friends to see! In this workshop, participants will explore the basics of Node.js and Express by making a HackerLog web app. You’ll guide them through setting up their JavaScript development environments, installing Visual Studio Code, downloading and testing an app locally, and seamlessly deploying it with Microsoft Azure.

ICON

Icon is an annual journal published with the collection of final year project report memo of both final year BCT & BEX students of Paschimanchal Campus. The first Icon had around 50 copies printed which included final year projects and memo of BCT/069 batch . The Icon vol-2 had combined the final year project of both BCT/070 and BEX/070. It included a total of 16 projects in which 8 were from BCT and 8 were from BEX. Icon is all about giving the idea for the projects to the next batch students and enhancing the current final batch students to do better and prosper their thoughts on their projects.
